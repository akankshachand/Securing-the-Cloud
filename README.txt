READ ME:

To run the project:

1. Run CloudGroup.py
2. Enter the team name you want to manage (TeamA, TeamB or TeamC unless you want to create a new group)
3. Run Users.py
4. Enter the username of a user from the team chosen above
5. Perfrom desired operations

The files to be uploaded on the drive need to be in the 'testfiles' folder.
The files downloaded from the drive will be downloaded in the 'downloads' folder.
